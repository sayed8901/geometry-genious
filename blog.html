With the HTML DOM, JavaScript can access and change all the elements of an HTML document. When a web page is loaded, the browser creates a Document Object Model (DOM) of the whole web page. The HTML DOM model is constructed as a tree of the Objects.
DOM can change or manipulate such as add or remove dynamically HTML objects. With DOM, we can also add or remove any attributes of literelly any HTML elements.

We can get HTML elements from DOM by using DOM methods. There are 5 major methods for getting an element from DOM. These are:
1. getElementsByTagName,
2. getElementbyId,
3. getElementsByClassName,
4. querySelector &
5. querySelectorAll.
GetElementbyId only works by ID name. On the other hand, getElementsByClassName getElementsByTagName or querySelectorAll provides an array-like-object. We can loop through it to get our desired element or output.

Event bubbling is a sequence of calling the event handlers. It is performed when one element is nested with another, and both elements have registered listeners for the same event.
If an event handler is set for the element, the event handler is triggered. Then the event "bubbles up" to the top as well as to the element's parent. The event continues to bubble, until it reaches the top (the document element).

A callback is a function passed as an argument to another function This technique allows a function to call another function. JavaScript functions are executed in the sequence they are called.
By this process, currently active or running function can use previously formed any other functions as needed. It is very useful regarding usage of re-usable functions.
